## üìû Troubleshooting

### "The pages won't load"
- Make sure all .html files are in the same directory as your other files
- Open in Firefox or Chrome (not IE)
- Check browser console (F12) for errors

### "Copy button doesn't work"
- Make sure you're using a modern browser
- Try Chrome or Firefox
- Check if clipboard API is enabled

### "Pages look broken on mobile"
- This is normal while resizing‚Äîrefresh after
- The responsive design adapts automatically
- Try on actual phone for better test

### "Demo button doesn't open demo"
- Make sure demo.html is in same directory
- Check that link paths are correct
- Clear browser cache if needed

### "Links to markdown files don't work"
- GitHub markdown files need to be accessed via GitHub raw content
- Or set up a simple web server
- Or convert markdown to HTML versions
